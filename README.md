# .github

Default community health files for all repositories under
[@kitsuyui](https://github.com/kitsuyui). A repository's own file, when
present, takes precedence over these defaults.

[@kitsuyui](https://github.com/kitsuyui) 配下の全リポジトリに適用される
デフォルトの community health files です。各リポジトリに同種のファイルが
ある場合はそちらが優先されます。

- [CONTRIBUTING.md](CONTRIBUTING.md) / [CONTRIBUTING.ja.md](CONTRIBUTING.ja.md)
- [AI_POLICY.md](AI_POLICY.md) / [AI_POLICY.ja.md](AI_POLICY.ja.md)
- [SECURITY.md](SECURITY.md)
- Issue forms / PR template: [.github/](.github/)
